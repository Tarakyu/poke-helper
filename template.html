⚡ 포켓몬 챔피언스 스피드 순위

싱글배틀 · 시즌 M-3 사용률 상위 100 · 스피드 빠른 순 · 출처 champs.pokedb.tokyo
계산 기준
  • 포켓몬 챔피언스 능력포인트(0~32) 시스템. 계산식: floor(floor((종족값×2+31+포인트×2)×50/100)+5) → 성격보정
  • 기본값 = 채용률 1위 포인트 세팅 + 채용률 1위 성격. 메가진화 나이트 1위인 경우 메가폼 종족값·특성 기준.
  • 능력포인트 직접 입력 / ▲●▼ 버튼으로 성격 보정 조정 가능.
